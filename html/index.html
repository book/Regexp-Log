Regexp::Log
Regexp::Log on CPAN | Distributions

Presentation

The modules that inherit from Regexp::Log can generate regular expressions for parsing specific log files.

This can be very useful for any application that allows to define a "custom" log format.

Modules inheriting from Regexp::Log

Only one module exists for the moment, but I plan to add several other. I am looking for help to refine the regular expressions, and to add support for other software.
